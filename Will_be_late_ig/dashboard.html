ML Pipeline

Every step from raw CSV to live address-based prediction.

01
Load & Inspect
Read 45,584 Zomato orders with 20 columns including coordinates, agent info, timestamps, weather.
df = pd.read_csv('Zomato_Dataset.csv')
02
Clean Missing Values
NaN in Age (1,854), Ratings (1,908), Weather (616). Median for numbers, 'Unknown' for text.
df.fillna(df.median()) | df[col].fillna('Unknown')
03
Engineer Distance
Haversine formula on restaurant/delivery lat-lon gives real km. Average: 9.3 km per order.
haversine(lat1,lon1,lat2,lon2) → km
04
Engineer Time Features
Pickup wait, order_hour, is_peak_hour (12–14h, 19–22h), day_of_week, is_weekend.
is_peak = 1 if hour in [12..14, 19..22]
05
Food Complexity Score
Domain knowledge: Drinks=1, Snack=2, Meal=3, Buffet=4. Longer prep = higher score.
{'Meal':3,'Buffet':4,'Snack':2,'Drinks':1}
06
Label Encode Categoricals
LabelEncoder on Weather, Traffic, City, Vehicle, Festival, Food type → integers. Saved to encoders.pkl.
le.fit_transform(df[col]) → encoders.pkl
07
Define Target Variable
is_late = 1 if delivery > 32 min (75th percentile of all orders). 24.7% of orders are late.
is_late = (Time_taken > 32).astype(int)
08
80/20 Stratified Split
Train: 36,467. Test: 9,117. Stratified to preserve the 24.7% late class ratio perfectly.
train_test_split(..., stratify=y)
09
Train 4 Models
Logistic Regression, Linear Regression, Random Forest (200 trees), XGBoost (300 trees).
RF → 92.7% | XGB → 92.5% | LR → 83.8%
10
Save Best Model
Random Forest saved as best_model.pkl with encoders.pkl and scaler.pkl for inference.
pickle.dump(model, open('best_model.pkl','wb'))
11
Address Geocoding
User types address → Nominatim (free OpenStreetMap API) converts it to lat/lon coordinates.
GET nominatim.openstreetmap.org/search?q=...
12
City Match + Delivery Radius
Closest city found by Haversine distance → restaurants within 15 km filtered → Random Forest predicts lateness for each.
POST /api/geocode → restaurants + predictions
